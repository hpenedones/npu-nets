# - The XDNA2 NPU path additionally requires:
#     * XRT under /opt/xilinx/xrt
#     * an editable IRON checkout, e.g. python -m pip install -e /path/to/IRON
# - mlir-aie wheel availability is platform-specific; follow the AMD / MLIR-AIE
#   installation instructions if your environment cannot resolve it directly.
